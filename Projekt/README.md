# Projekt: Algorytm szyfrowania RSA w języku Python

Projekt jest implementacją algorytmu szyfrowania RSA w języku Python, pozwala on zaszyfrowac tekst oraz go odszyfrować


## Uruchomienie

```
python3 szyfrowanie.py
```
